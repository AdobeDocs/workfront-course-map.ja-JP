# 新バージョンの Workfront での計画担当者の基本です。

プロジェクトの管理を初めて学ぶ際は、Workfront で物事をシンプルに保つことをお勧めします。Workfront でプロジェクトを作成、計画、管理、終了する方法について説明します。
