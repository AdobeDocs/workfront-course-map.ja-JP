# Workfront 管理者として学習を続ける

Workfront 管理者として学習を続ける Workfront One 学習パスは、Experience League の[システム設定に関する知識を深める（Workfront 管理者向け）](https://experienceleague.adobe.com/?recommended=Workfront-A-1-2022.2.admin){target="_blank"}に移行されました。

新しい Experience League コースは、以前の学習パスと同じ学習オブジェクティブをカバーし、可能な限り最適化され、包括的な学習エクスペリエンスを提供します。Experience League の Workfront コース、ドキュメント、チュートリアルおよび Communities のすべてのライブラリを参照してください。
